# ESoC 2026 Proposal: AgenticForecaster for sktime-mcp

**Applicant:** Saloni Sharma  
**GitHub:** [Saloni-0465](https://github.com/Saloni-0465)  
**Project area:** GC.OS + sktime STEP, sktime agentic/genAI  
**Proposal:** AgenticForecaster for sktime

This repository contains my European Summer of Code 2026 proposal for an agentic forecasting interface built on top of `sktime-mcp`.

AgenticForecaster is proposed as an LLM-driven, tool-calling forecasting agent that uses `sktime-mcp` tools as the execution layer, avoids transmitting raw time-series values to the LLM, evaluates a bounded set of candidate forecasting models, and returns explainable outputs with audit logs.

Full proposal: [proposal.md](proposal.md)
